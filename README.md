# Just Dip It Tacos — Website

Static site for justdipittacos.com, deployed via GitHub Pages.

## Pages
- `index.html` — Homepage
- `about.html` — About / Our Story
- `catering.html` — General catering menu + order form
- `just-dip-it-catering.html` — Corporate catering
- `contact.html` — Contact + order form

## Notes
- The order form on catering/contact pages is a static mockup, not yet wired to a real backend.
- Media files (photos/videos) referenced by filename must stay in this same root folder alongside the HTML.
